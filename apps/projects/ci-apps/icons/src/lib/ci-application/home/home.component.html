Adicione seus ícones a suas Aplicações

Com ferramentas que simplificam sistemas

Os ícones de projeto normalmente são arquivos .ico que contêm vários tamanhos1. Para especificar um ícone do aplicativo no Gerenciador de Soluções, escolha um nó do projeto (não o nó Solução) e, na barra de menus, escolha Projeto > Propriedades. Quando o Designer de Projeto for exibido, escolha a guia Aplicativo. Na lista Ícone, escolha um arquivo de ícone (.ico)2. Há vários sites que oferecem ícones gratuitos para qualquer projeto, como o Uicons3e o banco de dados de ícones gratuitos Ícones4.
